This is my personal website, built with SvelteKit and styled with a nostalgic GeoCities aesthetic. It features animated GIFs, visitor counters, guestbooks, and all the charming elements that made the early web so memorable.

## Features

- Retro design inspired by 1990s web aesthetics
- Built with modern SvelteKit for performance and developer experience
- Markdown-based content management for blog posts and projects
- Responsive layout that works on modern devices while maintaining vintage vibes

## Tech Stack

- **Framework**: SvelteKit
- **Styling**: CSS with retro design patterns
- **Content**: Markdown with frontmatter
- **Deployment**: Static site generation
